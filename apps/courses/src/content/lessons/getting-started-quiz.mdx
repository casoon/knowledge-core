# Quiz & Interaktivität

Eine der mächtigsten Features für Lernplattformen sind interaktive Quiz!

## Wie Quiz funktionieren

Quiz bestehen aus:
- **Fragen** mit mehreren Optionen
- **Richtige Antwort** (0-basierter Index)
- **Erklärungen** für die Lösung (optional)
- **Feedback** nach dem Check

<Callout type="info">
Nutzer können ihre Antworten überprüfen und erhalten sofortiges Feedback mit Erklärungen.
</Callout>

## Quiz in eigenen Kursen verwenden

So fügst du ein Quiz hinzu:

```mdx
import { Quiz } from '@knowledge-core/ui';

<Quiz
  title="Mein Quiz"
  questions={[
    {
      question: "Deine Frage?",
      options: ["Option 1", "Option 2", "Option 3"],
      correctAnswer: 1,
      explanation: "Erklärung der richtigen Antwort"
    }
  ]}
/>
```

<Hint title="Quiz-Best Practices">
- **Klare Fragen**: Stelle eindeutige Fragen
- **Plausible Optionen**: Alle Optionen sollten sinnvoll erscheinen
- **Hilfreiche Erklärungen**: Erkläre, warum die Antwort richtig ist
- **Angemessene Schwierigkeit**: Passe an deine Zielgruppe an
</Hint>
