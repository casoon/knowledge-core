## Platform Features

### 1. MDX-based Content

All content is written in **MDX** - Markdown with React components:

```mdx
# Heading

Normal text with **bold** and *italic*.

<Callout type="success">
Interactive component!
</Callout>
```

### 2. Interactive Elements

You can use components like Callouts, Quizzes, Exercises, and more.

### 3. Progress Tracking

The platform can track your learning progress (localStorage-based).

<Hint title="For Developers">
The entire source code is open source! You can customize and extend everything.
Check out the components in `packages/ui/src/components/`.
</Hint>
