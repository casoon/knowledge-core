# Interaktive Komponenten

Knowledge Core bietet verschiedene Komponenten für interaktive Inhalte.

## Callouts

Callouts heben wichtige Informationen hervor:

<Callout type="info" title="Information">
Dies ist eine Info-Callout für allgemeine Hinweise.
</Callout>

<Callout type="success" title="Erfolg">
Zeigt positive Bestätigungen oder erledigte Schritte.
</Callout>

<Callout type="warning" title="Warnung">
Wichtige Warnungen, die Nutzer beachten sollten.
</Callout>

<Callout type="error" title="Fehler">
Kritische Informationen oder häufige Fehler.
</Callout>

## Tabs

Tabs eignen sich für alternative Ansichten oder Multi-Platform-Anleitungen:

<Tabs tabs={[
  { label: 'Option A', value: 'a' },
  { label: 'Option B', value: 'b' },
  { label: 'Option C', value: 'c' }
]}>
  <TabPanel value="a">
    ### Option A

    Dies ist der Inhalt für Option A. Du kannst hier:
    - Text schreiben
    - Code-Beispiele zeigen
    - Weitere Komponenten nutzen
  </TabPanel>
  
  <TabPanel value="b">
    ### Option B

    Unterschiedlicher Inhalt für Option B.

    ```javascript
    const example = "Code-Beispiel in Tab B";
    console.log(example);
    ```
  </TabPanel>
  
  <TabPanel value="c">
    ### Option C

    <Callout type="info">
    Du kannst sogar Callouts in Tabs verschachteln!
    </Callout>
  </TabPanel>
</Tabs>

## Code-Blöcke

Syntax-Highlighted Code mit verschiedenen Sprachen:

**JavaScript:**
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

**TypeScript:**
```typescript
interface User {
  name: string;
  age: number;
}

const user: User = {
  name: 'Alice',
  age: 30
};
```

**Python:**
```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```
